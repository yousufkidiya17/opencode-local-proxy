Integrate Into Your Editors

Configure your development environment to consume AI directly from your local PC gateway.

Open your Continue config.json file (usually at ~/.continue/config.json) and add a custom model configuration:

{
  "models": [
    {
      "title": "Local DeepSeek Free",
      "provider": "openai",
      "model": "opencode/deepseek-v4-flash-free",
      "apiBase": "http://localhost:4000/v1",
      "apiKey": "aetherix-sk-master-9f3a7b2e1d"
    },
    {
      "title": "Local Big Pickle",
      "provider": "openai",
      "model": "opencode/big-pickle",
      "apiBase": "http://localhost:4000/v1",
      "apiKey": "aetherix-sk-master-9f3a7b2e1d"
    }
  ]
}

Configure Cursor to redirect completion calls to your local gateway:

  1. Go to Cursor Settings > Models.
  2. Scroll down to OpenAI API Key.
  3. Click on **Override OpenAI Base URL**.
  4. Set the URL to: http://localhost:4000/v1.
  5. Enter the Local API key: aetherix-sk-master-9f3a7b2e1d.
  6. Add your desired model IDs: opencode/deepseek-v4-flash-free and opencode/big-pickle.

Send prompts using the standard openai Python package:

from openai import OpenAI

client = OpenAI(
    base_url="http://localhost:4000/v1",
    api_key="aetherix-sk-master-9f3a7b2e1d"
)

response = client.chat.completions.create(
    model="opencode/deepseek-v4-flash-free",
    messages=[
        {"role": "user", "content": "Write a python quicksort script."}
    ],
    temperature=0.7,
    max_tokens=2048
)

print(response.choices[0].message.content)

Query the gateway using the standard JavaScript openai SDK:

import OpenAI from "openai";

const client = new OpenAI({
  baseURL: "http://localhost:4000/v1",
  apiKey: "aetherix-sk-master-9f3a7b2e1d"
});

const response = await client.chat.completions.create({
  model: "opencode/deepseek-v4-flash-free",
  messages: [{ role: "user", content: "Tell me a joke." }],
  temperature: 0.7,
  max_tokens: 2048
});

console.log(response.choices[0].message.content);
